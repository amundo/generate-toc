# generate-toc.js

This JavaScript utility generates a Table of Contents (TOC) as an HTML page with a nested list structure. It is designed to help you create a navigable TOC for your documentation or website.

here are what the flags mean:

    --force: Reinstalls the application if it already exists.
    --unstable: Allows the use of unstable Deno APIs.
    --allow-write: Grants permission to write files to the filesystem.
    --allow-read: Grants permission to read files from the filesystem.
    -n app: Names the installed application "app".
    ./app.ts: Specifies the path to the script to be installed.
